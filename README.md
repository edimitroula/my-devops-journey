## Data Security & Best Practices

**Important:** This repo excludes sensitive data files via `.gitignore`:
- Excel/Word files (`.xlsx`, `.docx`) - contain internal company data
- PDF documents - may contain confidential information
- Temporary/output files

**Only committed:**
- ✅ Python scripts (the automation tools)
- ✅ `requirements.txt` (dependencies)
- ✅ Documentation (this README)
- ✅ `.gitignore` (security rules)

**Store data files separately** in your organization's secure document repository (SharePoint, Confluence, etc.)
